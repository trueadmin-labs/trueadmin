# TrueAdmin Web

Web 目录用于承载 React 管理端应用。

## 技术选择

- React。
- Vite。
- TypeScript。
- Ant Design。

## 推荐结构

```text
src/
  app/
  pages/
  features/
  shared/
  services/
  stores/
```

## 第一阶段 Web 能力

- 登录页。
- 基础布局。
- 动态菜单。
- 权限路由。
- 用户管理。
- 角色管理。
- 菜单管理。
- 部门管理。
- 岗位管理。
- 字典管理。
- 日志管理。
- 文件上传。
- 个人中心。
- 基础仪表盘。

## 页面约定

- 列表页包含查询表单、表格、分页和操作列。
- 新增和编辑表单尽量复用。
- 删除操作需要二次确认。
- 按钮显示必须受权限点控制。
- API 请求统一放在 `services/` 或业务模块服务文件中。

## AI 开发提醒

AI 生成页面时，应先读取 API 契约和权限点，不要根据页面名称猜测字段。
