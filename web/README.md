# TrueAdmin Web

TrueAdmin Web 是基于 Ant Design Pro / Umi Max 初始化的后台管理端。

## 技术栈

- React 19
- TypeScript
- Ant Design 6
- Ant Design Pro Components
- Umi Max

## 常用命令

```bash
npm install
npm run dev
npm run tsc
npm run build
```

开发环境默认将 `/api/` 代理到 `http://127.0.0.1:9501`。

## 登录

默认后端种子账号：

- 用户名：`admin`
- 密码：`trueadmin`
